# Portfolio

A clean and simple portfolio for Tech Ambit Task

## Instructions

### Setup

If you use [nvm](https://github.com/nvm-sh/nvm) or [fnm](https://github.com/Schniz/fnm) (optional) -

```shell
nvm install
nvm use
```

Or -

```shell
fnm install
fnm use
```

Install and run -

```shell
yarn
yarn start
```

### In case of error

```shell
nvm install node --reinstall-packages-from=node
yarn install
yarn start
```
